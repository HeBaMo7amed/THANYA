# THANYA - Smart IoT-Based Emergency & Health Monitoring System

THANYA is a smart healthcare and emergency response web platform designed to improve response time during emergencies by providing fast and secure access to patient medical data and connected devices.

The system supports multiple user roles including Guest, Patient, and Paramedic.

---

## Live Demo
https://thanya-silk.vercel.app/

---

## My Role
Front-End Developer responsible for building the full web interface, user flows, and role-based dashboards.

---

## Key Features
- Multi-role system (Guest / Patient / Paramedic)
- Medical data management
- Emergency SOS tracking system
- Device monitoring and vital signs
- QR-based emergency access for paramedics
- Read-only secure access for emergency cases
- Store with cart and Stripe payment integration
